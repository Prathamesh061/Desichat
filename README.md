# DESIGRAM  

Desigram is a Social Media Application build with MERN Stack

### Features
- Login/Register
- Edit Profile
- Profile Page
- Friend Request
- Search
- Create Posts with text, images, video
- Comment, Like, Share Posts
- Notifications
- Explore
- Follow Suggestions
- Chat Capability

### Setup

```
cd desigram  
npm install  
cd client  
npm install  
cd ..  
npm run start  
```

Server runs on http://localhost:5555 and client on http://localhost:3000
